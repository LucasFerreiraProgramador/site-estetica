# Harmonização Facial - Site Estética

Bem-vindo ao projeto **Harmonização Facial**! Este é um site moderno e responsivo desenvolvido para apresentar serviços de estética facial, com foco em design elegante, experiência do usuário e performance.

## 🚀 Demonstração

Acesse o site: [https://lucasferreiraprogramador.github.io/site-estetica/](https://lucasferreiraprogramador.github.io/site-estetica/)

## ✨ Destaques do Projeto

- **Design Responsivo:** Layout adaptável para todos os dispositivos (desktop, tablet e mobile).
- **Visual Profissional:** Utilização de imagens de alta qualidade e paleta de cores sofisticada.
- **Performance:** Carregamento rápido e otimizado para web.
- **HTML5 & CSS3:** Código limpo, sem frameworks, focado em boas práticas e acessibilidade.
- **Fácil Customização:** Estrutura simples para personalização de conteúdo e imagens.

## 📁 Estrutura do Projeto

```
index.html         # Página principal
style.css          # Estilos customizados
assets/            # Imagens e recursos visuais
```

## 💡 Sobre o Projeto

Este site foi criado como portfólio para demonstrar habilidades em desenvolvimento front-end, design UI/UX e atenção aos detalhes. Ideal para clínicas de estética, profissionais de beleza ou qualquer negócio que deseje uma presença online marcante.
